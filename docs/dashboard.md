# Dashboard Notes

The current dashboard exists to make the local JobPipe state easier to review and act on.

It should help answer:

- what is worth pursuing now
- what needs follow-up
- what looks risky or noisy
- how the current evaluation flow is behaving

It should not define the product category by itself.

JobPipe is a career intelligence workbench and job-search decision system first. The dashboard is one projection layer over that workflow.
